# Sales-Data-Analysis-Python

## Project Overview
This project performs exploratory data analysis (EDA) on a sales dataset to extract meaningful business insights using Python.

## Technologies Used
- Python
- Pandas
- Matplotlib

## Key Features
- Data cleaning and preprocessing
- Monthly sales trend analysis
- Category-wise average sales and profit
- Product-wise and region-wise analysis
- Data visualization using charts

## Dataset
The dataset contains sales information including:
- Date
- Product
- Category
- Region
- Sales
- Profit

## How to Run
1. Install required libraries:
   pip install pandas matplotlib
2. Run the script:
   python sales_analysis.py

## Outcome
The analysis helps identify top-performing categories, sales trends over time, and profitable regions to support data-driven decision making.
